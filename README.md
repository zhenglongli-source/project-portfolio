# Technical Project Portfolio

This repository serves as a curated overview of my technical projects in applied mathematics, computational geometry, high-performance computing, optimization, and data analysis.

The goal of this portfolio is to make selected academic and independent projects easier to review, reproduce, and connect to my resume. Each project will include a clear description, code or reproducible materials when possible, example usage, results, and limitations.

## Featured Project Areas

### Computational Geometry

Projects related to polygon guarding, triangulation, visibility, and geometric algorithms.  
The long-term goal is to turn selected course-based work into a more complete and reusable computational geometry toolkit.

### High-Performance Computing

Projects involving MPI programming, parallel computing experiments, performance measurement, and cluster-based workflows.

### Portfolio Optimization

Projects related to quantitative modeling, portfolio construction, and possible trading-platform extensions.

### Numerical Methods and Data Analysis

Course-based and independent computational projects involving numerical methods, regression, simulation, and applied data analysis.

### AI Workflow Agent

An experimental project idea involving coordination between stronger and weaker language models to improve cost efficiency for large-scale or repetitive AI-assisted workflows.
